# Alugames  - Projeto de Gerenciamento de Aluguel de Jogos

Este projeto é um exercício de programação de lógica para gerenciar o aluguel de jogos usando Javascipt. Ele permite adicionar, listar, atualizar e excluir jogos, bem como registrar aluguéis e devoluções.

Este projeto/exercício é fornecido pela Alura como parte do curso Lógica de Programação.
